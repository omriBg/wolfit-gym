# Wolfit Gym - אימונים חכמים במתחם 🏋️‍♂️

## מה זה?
מערכת חכמה שעוזרת לכם לתכנן אימונים במתחם הספורט שלנו:
- בונה אימון לפי המגרשים והמתקנים הפנויים
- מתאימה את האימון להעדפות הספורט שלכם
- מייצרת תוכנית אופטימלית לפי הזמינות במתחם

## איך זה עובד?
1. בוחרים את סוגי הספורט המועדפים עליכם
2. המערכת מוצאת את המגרשים הפנויים המתאימים
3. מקבלים תוכנית אימון מותאמת אישית! 💪

## נתקלתם בבעיה?
ספרו לנו:
- מה לא עובד?
- באיזה מסך?
- צילום מסך
- באיזה מכשיר? (טלפון/מחשב)
