# אנימציה למסך MainMenu

## תיאור
האנימציה כוללת שלושה שלבים:
1. **Soccer 1** - אנימציה של כדורגל עם קווי מתאר ומילוי
2. **Soccer 2** - אנימציה של כדורגל נוסף עם אפקטים שונים
3. **Basket** - אנימציה של סל עם קווי מתאר ומילוי

## ספריות נדרשות
- **GSAP** (GreenSock Animation Platform) - לאנימציות מתקדמות
- **Anime.js** - לאנימציות SVG מתקדמות

## התקנה
```bash
npm install
```

## שימוש
האנימציה תופיע אוטומטית כאשר המסך נטען ותיעלם בסיום.

## מבנה הקוד
- `AnimationComponent.js` - הקומפוננטה הראשית של האנימציה
- `AnimationComponent.css` - הסגנונות לאנימציה
- `MainMenu.js` - המסך הראשי שכולל את האנימציה

## הוספת תוכן SVG
כדי להוסיף את התוכן האמיתי של האנימציה, יש להחליף את ההערות ב-SVG עם התוכן האמיתי:

```jsx
<g id="soccer1">
  <g className="soccer1_extra-line">
    {/* הוסף כאן את הקווי המתאר הנוספים */}
  </g>
  <g className="soccer1_line">
    {/* הוסף כאן את הקווי המתאר הראשיים */}
  </g>
  <g className="soccer1_fill">
    {/* הוסף כאן את אלמנטי המילוי */}
  </g>
  <g className="soccer1ball">
    <g className="soccer1ball-line">
      {/* הוסף כאן את קווי הכדור */}
    </g>
    <g>
      {/* הוסף כאן את אלמנטי הכדור */}
    </g>
  </g>
</g>
```

## התאמה אישית
- ניתן לשנות את זמני האנימציה על ידי עריכת ערכי `duration` ו-`delay`
- ניתן לשנות את סוגי האנימציה על ידי עריכת ערכי `easing`
- ניתן להוסיף אנימציות נוספות על ידי הוספת פונקציות חדשות

## פתרון בעיות
אם האנימציה לא עובדת:
1. ודא שכל הספריות הותקנו כראוי
2. בדוק את הקונסול לשגיאות JavaScript
3. ודא שתוכן ה-SVG הוסף כראוי
