# הוספת בחירת אזורי גוף לאימון כוח

## סקירה כללית
הוספנו פיצ'ר חדש שמאפשר למשתמשים לבחור אם הם רוצים לשלב תרגילי כוח באימון שלהם, ובמקרה שכן - לבחור איזה אזורי גוף הם רוצים לעבוד עליהם.

## שינויים שבוצעו

### 1. Frontend - מסך עריכת משתמש (EditUser.js)
- ✅ הוספת state חדש: `wantsStrengthTraining` ו-`selectedBodyAreas`
- ✅ הוספת רשימת אזורי גוף: גב, כתפיים, ידיים, חזה, ליבה/בטן, רגליים
- ✅ הוספת ממשק משתמש עם checkbox ו-grid של כפתורי אזורי גוף
- ✅ עדכון פונקציות טעינה ושמירה

### 2. Frontend - מסך הרשמה (SignUpPreferences.js)
- ✅ הוספת אותם שדות כמו במסך עריכת משתמש
- ✅ הוספת אותו ממשק משתמש
- ✅ עדכון פונקציית ההרשמה

### 3. Frontend - עיצוב (CSS)
- ✅ הוספת עיצוב לשדה אימון כוח
- ✅ עיצוב לכפתורי אזורי גוף עם אנימציות
- ✅ התאמה למובייל ומסכים שונים

### 4. Backend - סכמת מסד נתונים
- ✅ הוספת שדות חדשים לטבלת User:
  - `wantsStrengthTraining BOOLEAN DEFAULT FALSE`
  - `selectedBodyAreas TEXT[] DEFAULT '{}'`
- ✅ יצירת קובץ SQL לעדכון מסד נתונים קיים

### 5. Backend - שרת (server.js)
- ✅ עדכון סכמת וולידציה לכלול שדות חדשים
- ✅ עדכון פונקציית שמירת העדפות
- ✅ עדכון פונקציית טעינת העדפות
- ✅ עדכון פונקציית הרשמה

## הוראות הפעלה

### 1. עדכון מסד הנתונים
הרץ את הקובץ SQL הבא על מסד הנתונים שלך:
```sql
-- הרץ את הקובץ: add_body_areas_fields.sql
```

### 2. הפעלת השרת
```bash
cd backend
npm start
```

### 3. הפעלת הפרונט
```bash
cd frontend
npm start
```

- 🦾 ידיים

- 🎯 ליבה/בטן

## תכונות
- ✅ המשתמש יכול לבחור אם הוא רוצה אימון כוח
- ✅ אם בחר כן, יפתח לו grid של אזורי גוף לבחירה
- ✅ יכול לבחור אחד או יותר אזורי גוף
- ✅ הנתונים נשמרים במסד הנתונים
- ✅ הנתונים נטענים בעת עריכת המשתמש
- ✅ עיצוב רספונסיבי למובייל ודסקטופ

## מבנה הנתונים
```json
{
  "wantsStrengthTraining": true,
  "selectedBodyAreas": ["back", "shoulders", "arms"]
}
```

## הערות טכניות
- השדות החדשים הם אופציונליים ולא ישפיעו על משתמשים קיימים
- הוולידציה מבטיחה שרק אזורי גוף חוקיים יישמרו
- העיצוב תואם לעיצוב הקיים של האפליקציה
